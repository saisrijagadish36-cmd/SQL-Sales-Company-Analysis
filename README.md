# 📊 SQL Sales Company Analysis

## 📌 Project Overview
This project analyzes company sales data using SQL.  
The objective is to extract meaningful business insights such as total revenue, regional performance, product performance, and monthly trends.

---

## 🛠 Tools & Technologies Used
- MySQL
- SQL Aggregation Functions (SUM, AVG, COUNT)
- GROUP BY
- ORDER BY
- Subqueries

---

## 🗂 Database Structure

The sales table contains the following columns:

- id (Primary Key)
- order_date
- product
- region
- quantity
- price

---

## 📈 Business Questions Solved

1. What is the total company revenue?
2. Which region generates the highest revenue?
3. What is the monthly sales performance?
4. Which product generates the most revenue?
5. Which orders are above average value?

---

## 📊 Key Insights

- Highest revenue region: North (₹375,000)
- Best-performing product: Laptop (₹450,000)
- Peak sales month: January (₹495,000)
- Average Order Value: ₹110,000
